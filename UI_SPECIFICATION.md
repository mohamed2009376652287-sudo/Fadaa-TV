# مواصفات واجهة المستخدم - تطبيق الفضائي TV

هذا الملف يحدد الهيكل التصميمي المعتمد للتطبيق لضمان احترافية وتجربة مستخدم ممتازة.

## 1. التخطيط العام (Layout)
- **النمط:** Dark Mode (خلفية داكنة مع تباين ألوان احترافي).
- **التنقل السفلي (Bottom Navigation):**
  - الرئيسية (Home)
  - البطولات (Leagues)
  - البث المباشر (Live Stream)
  - الأخبار (News)
  - المزيد (More)

## 2. شاشة البث المباشر (Watch Screen)
يتم ترتيب العناصر من الأعلى للأسفل كالتالي:
1. **معلومات المباراة:** اسم البطولة، الوقت، شعار الفريقين، النتيجة.
2. **مشغل الفيديو:** في منتصف الشاشة مع أزرار التحكم والجودات.
3. **أدوات المساعدة:** أزرار اختيار السيرفرات واختيار اللغة (AR, FR, EN).
4. **تبويبات البيانات (Tabs):** - التفاصيل (إحصائيات المباراة: الاستحواذ، التسديدات، التمريرات).
   - التشكيلة.
   - الترتيب.
5. **معلومات إضافية:** الحكم، الملعب، القناة الناقلة.

## 3. صفحة "المزيد" (More Screen)
تحتوي على:
- الدعم الفني.
- مشاركة التطبيق.
- الإعدادات.
- حول التطبيق.
